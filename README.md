# COVID-19 Epidemiology Knowledge Graph

**Predict infection hotspots and symptom patterns using a synthetic COVID-19 Knowledge Graph built with Python. Research-ready & beginner-friendly.**

---

## Problem
COVID-19 spread is complex, influenced by patient locations, events, and symptoms. Traditional tabular data fails to model relationships for predictive insights. This project constructs a Knowledge Graph to enable predictive analysis of hotspots and symptom links.

## Solution
- Build a Knowledge Graph from synthetic datasets: **Patients, Cities, Events, Symptoms**  
- Supports ML tasks: **link prediction**, **hotspot prediction**, **symptom co-occurrence analysis**  
- Fully reproducible in Python using **NetworkX**, with CSV datasets for experimentation  

## Key Features
- End-to-end Knowledge Graph construction in Python  
- Visualization-ready with NetworkX & Matplotlib  
- Supports ML experiments for infection hotspot prediction  
- Scalable synthetic dataset (~9,000 relationships)  
- Beginner-friendly, research-ready, reproducible
